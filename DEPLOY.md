# 📦 Instructions de déploiement GitHub + Vercel

## Étape 1 : Push sur GitHub

### Via GitHub Web (le plus simple sur Android)

1. Va sur https://github.com/new
2. Nom du repo : `bob-presentation`
3. Public ou Private : **Public** (pour Vercel gratuit)
4. Ne coche RIEN (pas de README, pas de .gitignore)
5. Clique "Create repository"

6. Une fois créé, clique sur "uploading an existing file"
7. Drag & drop ou sélectionne :
   - `index.html`
   - `ymani_audio.mp3`
   - `README.md`
8. Commit message : "Initial commit - Présentation interactive"
9. Clique "Commit changes"

✅ Ton repo GitHub est prêt !

---

## Étape 2 : Déployer sur Vercel

1. Va sur https://vercel.com
2. Clique "Sign Up" → Connecte-toi avec GitHub
3. Une fois connecté, clique "Add New..." → "Project"
4. Cherche ton repo `bob-presentation`
5. Clique "Import"
6. Laisse tout par défaut et clique "Deploy"

⏳ Attends 30 secondes...

🎉 **C'EST EN LIGNE !**

Vercel te donne un lien genre : `bob-presentation.vercel.app`

---

## Étape 3 : Partager

Tu peux maintenant :
- Partager le lien Vercel directement
- L'ajouter dans Canva comme bouton/lien
- Le mettre dans ton LinkedIn

---

## 🔄 Pour mettre à jour la présentation plus tard

Sur GitHub :
1. Va dans ton repo
2. Clique sur le fichier à modifier
3. Clique sur l'icône ✏️ (Edit)
4. Fais tes modifs
5. "Commit changes"

Vercel redéploiera automatiquement ! ✨
